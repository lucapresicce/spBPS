# Gibbs sampler for Conjugate Bayesian Multivariate Linear Models

Gibbs sampler for Conjugate Bayesian Multivariate Linear Models

## Usage

``` r
bayesMvLMconjugate(Y, X, mu_B, V_B, nu, Psi, n_iter = 1000, burn_in = 500)
```

## Arguments

- Y:

  [matrix](https://rdrr.io/r/base/matrix.html) \\n \times q\\ of
  response variables

- X:

  [matrix](https://rdrr.io/r/base/matrix.html) \\n \times p\\ of
  predictors

- mu_B:

  [matrix](https://rdrr.io/r/base/matrix.html) \\p \times q\\ prior mean
  for \\\beta\\

- V_B:

  [matrix](https://rdrr.io/r/base/matrix.html) \\p \times p\\ prior row
  covariance for \\\beta\\

- nu:

  [double](https://rdrr.io/r/base/double.html) prior parameter for
  \\\Sigma\\

- Psi:

  [matrix](https://rdrr.io/r/base/matrix.html) prior parameter for
  \\\Sigma\\

- n_iter:

  [integer](https://rdrr.io/r/base/integer.html) iteration number for
  Gibbs sampler

- burn_in:

  [integer](https://rdrr.io/r/base/integer.html) number of burn-in
  iteration

## Value

B_samples [array](https://rdrr.io/r/base/array.html) of posterior sample
for \\\beta\\

Sigma_samples [array](https://rdrr.io/r/base/array.html) of posterior
samples for \\\Sigma\\

## Examples

``` r
## Generate data
n <- 100
p <- 3
q <- 2
Y <- matrix(rnorm(n*q), nrow = n, ncol = q)
X <- matrix(rnorm(n*p), nrow = n, ncol = p)

## Prior parameters
mu_B <- matrix(0, p, q)
V_B <- diag(10, p)
nu <- 3
Psi <- diag(q)

## Samples from posteriors
n_iter <- 1000
burn_in <- 500
set.seed(1234)
samples <- spBPS::bayesMvLMconjugate(Y, X, mu_B, V_B, nu, Psi, n_iter, burn_in)
```
